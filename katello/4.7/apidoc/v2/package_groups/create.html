POST /katello/api/package_group
Create a package group

Examples

POST /katello/api/package_groups
{
  "repository_id": 367202338,
  "name": "My_Group",
  "description": "My Group",
  "mandatory_package_names": [
    "katello-agent"
  ],
  "package_group": {
    "name": "My_Group",
    "description": "My Group"
  }
}
200
{
  "status": "success"
}
